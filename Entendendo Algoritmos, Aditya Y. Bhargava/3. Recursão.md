- A segunda maneira utiliza a **recursão**. 
- ***Recursão*** é quando uma função chama a si mesma. Vejamos:

    def procure_pela_chave(caixa):

        for item in caixa:

            if item.e_uma_caixa():

                procure_pela_chave(item)

            elif item.e_uma_chave():

                print "Encontrei a chave"


- A recursão é usada para tornar a resposta mais *clara*.

- Os loops podem melhorar o desempenho do programa, mas a função melhora o desempenho do programador.

- Todas as funções recursivas têm duas partes : $o caso-base e o caso recursivo$. 
  - O **caso recursivo** é quando a função chama a si mesma.
  - O **caso-base** é quando a função não chama a si mesma novamente, de forma a que o programa não se transforme num loop infinito.

- O computador utiliza uma pilha interna chamada de ***pilha de chamada***.

- Quando se chama uma função a partir de outra, a chamada de função fica pausada num estado parcialmente completo. Todos os valores das variáveis para aquela função ainda estão armazenados na memória.

- As funções recursivas também utilizam a pilha de chamada.

- Cada uma destas funções de chamada ocupa um pouco de memória, e quando a pilha está muito cheia é sinal de que o computador está a guardar informação para muitas chamadas de funções. Para esta opção há duas opções:
  - Reescrever o código utilizando loops.
  - Utilizar o que chamamos de *tail recursion* (recursão de causa).

- Recapitulando:
  - Recursão é quando uma função se chama a si mesma.
  - Uma pilha tem duas operações : $push e pop$.
  - Todas as chamadas de função vão para a pilha de chamada.
